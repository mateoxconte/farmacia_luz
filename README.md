# FARMACIA_TUP
# Farmacia Luz – Backend

Sistema de gestión desarrollado en C# y SQL Server, orientado a la administración
de productos y procesos internos. Proyecto académico con foco en backend,
lógica de negocio y manejo de datos.

## Tecnologías
- C# (.NET)
- SQL Server
- APIs REST
- Arquitectura en capas
- Git

## Funcionalidades
- ABM de entidades
- Validaciones de datos
- Control de estados
- Persistencia en base de datos relacional

## Arquitectura
Proyecto organizado en capas:
- Presentación / API
- Lógica de Negocio
- Acceso a Datos
